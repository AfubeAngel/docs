## Authorization

To begin using the TMDb API, you must first sign up for an account. If you do not have an account, registering for an account is free and simple. Click [here](https://www.themoviedb.org/signup) to get started.

Once you've registered, follow these steps to obtain your API key:

- Log in to your TMDb account.
- Visit your [Account Settings page](https://www.themoviedb.org/settings/api).
- Click on the API link to register for an API key.

_Please note that the API registration process is optimized for desktop devices. For the best experience, we recommend completing the registration on a desktop computer using a browser._
